性格

Mar 2, 2017 | Hits

首先人格形成受到两个因素的影响：nature（先天）以及nurture（后天养育）。先天很好理解，譬如父母都具有I的特制的话，这种气质就可能会遗传给孩子。后天的话，我个人认为影响并不如先天大。至于形成intj特质的原因，可能是孩童时期独处时间相对多，加上阅读等各方面的塑造吧。每个intj都是有差异的。
